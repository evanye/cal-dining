How about yesterday? Or only food at crossroads? Or only during lunch?

Make a GET request with params: /menu?date= ‹string:date› & meal= ‹string:meal› & location= ‹string:dining_hall›

Date

Use the string pattern "Year-Month-Day". For example: date=2013-06-01. Note The server only has data starting from June, 2013.

Date

Select a date range with start and end. For example: start=2013-06-01 & end=2013-06-10

Meal

Specify any combination of the three meals: breakfast, lunch, or dinner

Location

Specify any combination of the 3 dining halls: crossroads, foothill, cafe3, or clarkkerr

Food Information API

Make a GET request to /food.
